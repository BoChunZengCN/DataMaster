# 3D 产品数据在线预览工具

一个基于 Web 的专业 3D 模型预览工具，面向产品设计人员、营销人员和创作者。

## 在线体验

直接在浏览器中打开 `POC.html` 即可使用。

## 功能特性

### 文件导入
- 支持 **STL**、**OBJ**、**STEP**、**IGES** 格式
- 拖拽上传或点击选择文件
- 自动缩放至合适尺寸

### 3D 渲染
- 基于 Three.js 的 WebGL 渲染
- 支持旋转、平移、缩放操作
- 双面渲染，模型背面不消失

### 视图控制
- 7 种标准视图：前、后、左、右、顶、底、等轴测
- 自定义视图保存与快速切换
- 平滑的相机动画过渡

### 材质系统
- 12 种预设材质（金属银、黄金、碳纤维、陶瓷白等）
- 自定义材质上传（颜色、金属度、粗糙度、贴图）
- 实时材质切换

### 背景设置
- 纯色背景
- 渐变背景（默认）
- 自定义图片背景

### 测量工具
- 最短距离测量
- XYZ 轴距离测量
- 屏幕空间标签显示

### 截面分析
- X/Y/Z 轴截面
- 截面位置调整
- 截面厚度控制
- 截面方向反转

### 模型库
- 本地 IndexedDB 持久化存储
- 目录分类管理
- 模型跨目录移动

## 操作指南

| 操作 | 功能 |
|------|------|
| 左键拖拽 | 旋转视图 |
| 右键拖拽 | 平移视图 |
| 滚轮 | 缩放视图 |
| 点击 | 测量模式下选取点 |
| Esc | 退出测量模式 |

## 技术栈

- Three.js 0.160.0
- OpenCASCADE WebAssembly (occt-import-js)
- IndexedDB
- 纯前端实现，无需后端

## 浏览器兼容性

- Chrome / Edge / Firefox / Safari 最新版
- 需要 WebGL 支持

## 本地运行

```bash
# 使用 Python 启动本地服务器
python -m http.server 8080

# 然后访问 http://localhost:8080/POC.html
```

或者直接用浏览器打开 `POC.html` 文件。
